What is Poketch?

Poketch is a tool for tracking the progress of completing the so-called Living Pokédex in games in the Pokémon series. You can check this link, if you want to learn more about the Living Dex Challenge.

Do I need to create an account to use Poketch?

Yes, to store your Pokémon collection, you need to create a free Poketch account.

What are current features of Poketch?

  • Storing data about owned Pokémon
  • Statistics page: displays statistics of user's Pokémon collection
  • Box utility tool: for calculating position of Pokémon in their PC Box

Is Poketch open source?

Yes, you can get source code by visiting github.com/danrog303/poketch-spring.
